5 июля.
Добавить модуль Logging (и подключить, чтобы работал)
это система, которая в начале сохраняет в памяти состояние всех датчиков. Потом выводит в текстовый файл сводку изменений в таком виде (а файл текстовый, и все новые сводки дописываются ниже):

5 july 2018. 10:23am ===
 ON Sensor "ID, имя сенсора, тип сенсора"
 ON Sensor #23 Motion Входная дверь
 ON Sensor "ID, имя сенсора, тип сенсора"
...

то есть если при опросе всех датчиков в данный момент у несколькоик произошли изменения состояния с ОБНАРУЖИЛ-движение на НЕ ОБНАРУЖИЛ
или наоборот. Эти датчики должны быть показаны списком с их текущими состояниями.

как работать с файлами прочтете в документации Lua.
сначала можете выводить просто print(в консоль)

файл модуля : logging.lua
функция модуля начальной инициализации:
  init(имя файла лога)
функция модуля которая вызывается внутри каждого update каждого датчика:
 checkState()
функция которая вызывается в каждом love.update:
 updateLog()  (она же вызывает локальную функцию из модуля, которая пишет в файл. когда есть изменения.)
